# Product Requirements Document (PRD)

## 프로젝트 개요

### 제품명
축구 전술 전략 코치/감독용 웹 애플리케이션

### 버전
1.0.0 (MVP)

### 작성일
2024년

### 목적
TacticalPad를 레퍼런스로 하여 HTML5 Canvas 기반의 축구 전술 전략을 시각화하고 관리할 수 있는 웹 애플리케이션을 개발합니다.

---

## 목표 사용자

### 주요 사용자
- **축구 코치**: 훈련 세션 계획 및 전술 설명
- **축구 감독**: 경기 전술 수립 및 선수 배치
- **축구 분석가**: 전술 분석 및 시각화

### 사용자 특성
- 축구 전술에 대한 기본 지식 보유
- 웹 브라우저 사용 가능
- 전술을 시각적으로 표현하고 싶어함
- 훈련 세션을 체계적으로 관리하고 싶어함

---

## 핵심 가치 제안

### 문제 해결
1. **전술 시각화의 어려움**: 종이와 펜으로 전술을 그리는 것은 수정이 어렵고 공유가 불편함
2. **전술 관리의 비효율**: 여러 전술을 체계적으로 관리하기 어려움
3. **애니메이션 부재**: 정적인 전술도면으로는 움직임을 설명하기 어려움
4. **비용 문제**: TacticalPad와 같은 상용 소프트웨어는 구독 비용이 부담됨

### 솔루션
1. **직관적인 전술 보드**: 드래그 앤 드롭으로 쉽게 선수를 배치하고 전술을 그릴 수 있음
2. **세션 관리 시스템**: 훈련 세션별로 전술을 체계적으로 관리
3. **애니메이션 기능**: 선수의 움직임을 시각적으로 표현
4. **무료 웹 앱**: 브라우저만 있으면 사용 가능, 설치 불필요

## 기능 요약

### MVP 기능

#### 1. 전술 보드
- 축구장 배경 표시
- 선수 배치 및 이동 (드래그 앤 드롭)
- 전술 도형 그리기 (화살표, 선, 영역)
- 객체 선택 및 편집
- 줌/팬 기능

#### 2. 애니메이션
- 키프레임 기반 애니메이션 생성
- 애니메이션 재생/일시정지
- 타임라인 편집
- 선수 이동 경로 표시

#### 3. 세션 관리
- 세션 생성/편집/삭제
- 세션 목록 및 검색
- 전술 저장 및 불러오기
- 세션 썸네일

#### 4. 데이터 관리
- 자동 저장 (IndexedDB)
- 수동 저장
- JSON 내보내기/가져오기
- 이미지 내보내기

### 향후 기능 (Phase 2+)
- 포메이션 템플릿
- 격자 표시 및 스냅
- 실행 취소/다시 실행
- 키보드 단축키
- 협업 기능
- 클라우드 저장
- 모바일 앱

## 기술적 제약사항

### 플랫폼
- **웹 브라우저**: Chrome, Firefox, Safari, Edge (최신 2개 버전)
- **모바일**: 기본 지원 (향후 최적화)

### 기술 스택
- **프론트엔드**: React 18+ with TypeScript
- **Canvas**: React-Konva
- **상태 관리**: Zustand
- **스타일링**: Tailwind CSS
- **빌드 도구**: Vite
- **데이터 저장**: IndexedDB (로컬)

### 제약사항
- **로컬 저장만**: 초기 버전은 클라우드 저장 미지원
- **단일 사용자**: 협업 기능 없음
- **2D만 지원**: 3D 뷰는 향후 기능
- **오프라인**: 인터넷 연결 없이도 작동

## 성공 지표

### 기능적 지표
- [ ] 모든 MVP 기능 구현 완료
- [ ] 주요 유저 스토리 80% 이상 완료

- 애니메이션 재생: 60fps 유지

- 직관적인 UI로 학습 곡선 최소화
- 주요 작업 3클릭 이내 완료
- 사용자 만족도 조사 (향후)

## 우선순위

### P0 (필수 - MVP)
1. 전술 보드 기본 기능 (선수 배치, 도형 그리기)
2. 애니메이션 기본 기능 (재생, 타임라인)
3. 세션 생성 및 저장/불러오기
4. 로컬 저장소 연동

### P1 (높음 - Phase 2)
1. 포메이션 템플릿
2. 줌/팬 기능
3. 실행 취소/다시 실행
4. 이미지 내보내기

### P2 (중간 - Phase 3)
1. 격자 표시
2. 키보드 단축키
3. JSON 내보내기/가져오기
4. UI/UX 개선

### P3 (낮음 - 향후)
1. 협업 기능
2. 클라우드 저장
3. 모바일 앱
4. 3D 뷰

## 위험 및 가정

### 위험
1. **Canvas 성능 이슈**: 대량의 객체 처리 시 성능 저하 가능
   - **완화**: 레이어 분리, 최적화 기법 적용
2. **브라우저 호환성**: IndexedDB 지원 차이
   - **완화**: 폴백으로 localStorage 사용
3. **복잡한 애니메이션 로직**: 구현 난이도 높을 수 있음
   - **완화**: 단계적 구현, 라이브러리 활용

### 가정
1. 사용자는 최신 브라우저를 사용함
2. 사용자는 기본적인 웹 사용 능력을 보유함
3. 사용자는 축구 전술에 대한 기본 지식을 보유함
4. 초기 버전은 무료로 제공됨
