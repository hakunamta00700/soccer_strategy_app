# 기능 명세서 및 요구사항

## 프로젝트 개요

축구 전술 전략을 지원하는 코치/감독용 웹 애플리케이션으로, TacticalPad를 레퍼런스로 하여 HTML5 Canvas 기반으로 구현됩니다.

## 핵심 기능

### 1. 전술 보드 (Tactical Board)

#### 1.1 기본 전술판
- **축구장 배경**: 표준 축구장 레이아웃 (반원, 페널티 박스, 골대 등)
- **줌/팬 기능**: 전술판 확대/축소 및 이동
- **격자 표시**: 정확한 배치를 위한 격자 옵션
- **다양한 포메이션 템플릿**: 4-4-2, 4-3-3, 3-5-2 등 기본 포메이션 제공

#### 1.2 선수 배치 및 관리
- **선수 아이콘 배치**: 드래그 앤 드롭으로 선수 배치
- **선수 속성 설정**:
  - 등번호
  - 이름
  - 포지션 (GK, DF, MF, FW)
  - 팀 구분 (홈/어웨이, 색상)
- **선수 이동**: 배치된 선수를 자유롭게 이동
- **선수 복사/삭제**: 선수 복제 및 제거 기능
- **포메이션 자동 배치**: 선택한 포메이션에 따라 선수 자동 배치

#### 1.3 전술 도형 그리기
- **화살표**: 공의 이동 경로, 선수 움직임 표시
  - 단일 화살표
  - 곡선 화살표
  - 화살표 스타일 커스터마이징 (두께, 색상, 점선/실선)
- **선 그리기**: 전술 라인 표시
  - 직선
  - 곡선
  - 자유 곡선
- **영역 표시**:
  - 사각형 영역
  - 원형 영역
  - 다각형 영역
  - 반투명 채우기 옵션
- **텍스트 주석**: 전술 설명 텍스트 추가
- **도형 편집**: 선택한 도형의 속성 수정 (색상, 크기, 위치 등)
- **도형 삭제**: 선택한 도형 제거

### 2. 애니메이션

#### 2.1 애니메이션 생성
- **키프레임 설정**: 시간대별 선수 위치 및 도형 상태 저장
- **움직임 경로**: 선수 이동 경로 시각화
- **타임라인 편집**: 애니메이션 타임라인에서 키프레임 추가/수정/삭제
- **애니메이션 속도 조절**: 재생 속도 조절 (0.5x ~ 2x)

#### 2.2 애니메이션 재생
- **재생/일시정지**: 애니메이션 재생 제어
- **프레임 단위 이동**: 이전/다음 프레임으로 이동
- **구간 반복**: 특정 구간 반복 재생
- **재생 모드**: 
  - 정방향 재생
  - 역방향 재생
  - 왕복 재생

#### 2.3 애니메이션 관리
- **애니메이션 저장**: 전술에 애니메이션 저장
- **애니메이션 복사**: 다른 전술에 애니메이션 복사
- **애니메이션 삭제**: 저장된 애니메이션 제거

### 3. 세션 계획 (Session Planning)

#### 3.1 세션 생성 및 관리
- **세션 생성**: 새로운 훈련 세션 생성
- **세션 편집**: 세션 정보 수정 (이름, 설명, 날짜 등)
- **세션 삭제**: 세션 제거
- **세션 목록**: 저장된 세션 목록 표시 및 검색

#### 3.2 전술 관리
- **전술 생성**: 세션 내 새로운 전술 생성
- **전술 복사**: 기존 전술 복제
- **전술 삭제**: 전술 제거
- **전술 미리보기**: 전술 썸네일 표시

### 4. 데이터 관리

#### 4.1 로컬 저장
- **자동 저장**: 작업 내용 자동 저장 (IndexedDB)
- **수동 저장**: 사용자가 명시적으로 저장
- **저장 상태 표시**: 저장 완료/미저장 상태 표시

#### 4.2 데이터 불러오기
- **세션 불러오기**: 저장된 세션 불러오기
- **전술 불러오기**: 특정 전술 불러오기
- **최근 작업**: 최근 작업한 세션 목록

#### 4.3 내보내기/가져오기
- **JSON 내보내기**: 전술 데이터를 JSON 형식으로 내보내기
- **JSON 가져오기**: JSON 파일로부터 전술 데이터 가져오기
- **이미지 내보내기**: 전술 보드를 이미지로 저장 (PNG, JPG)
- **PDF 내보내기**: 전술 보드를 PDF로 저장 (향후 기능)

## 비기능 요구사항

### 성능
- **렌더링 성능**: 60fps 유지 (애니메이션 재생 시)
- **반응성**: 사용자 입력에 대한 즉각적인 피드백 (< 100ms)
- **메모리 관리**: 대량의 객체 처리 시 메모리 효율적 사용
- **Canvas 최적화**: 불필요한 리렌더링 최소화

### 사용성
- **직관적인 UI**: 학습 곡선 최소화
- **키보드 단축키**: 자주 사용하는 기능에 단축키 제공
- **반응형 디자인**: 다양한 화면 크기 지원 (웹 우선)
- **접근성**: 키보드만으로도 주요 기능 사용 가능

### 확장성
- **모듈화된 구조**: 기능별 모듈 분리
- **플러그인 아키텍처**: 향후 기능 확장 용이
- **다중 스포츠 지원**: 축구 외 다른 스포츠 지원 가능한 구조 (향후)

### 호환성
- **브라우저 지원**: 
  - Chrome (최신 2개 버전)
  - Firefox (최신 2개 버전)
  - Safari (최신 2개 버전)
  - Edge (최신 2개 버전)
- **모바일 브라우저**: 기본 지원 (향후 최적화)

### 데이터 보안
- **로컬 저장**: 모든 데이터는 사용자 로컬에만 저장
- **데이터 백업**: 사용자가 수동으로 백업 가능
- **개인정보 보호**: 사용자 데이터 수집 없음

## 우선순위

### MVP (Minimum Viable Product)
1. 기본 전술 보드 (축구장 배경, 선수 배치)
2. 기본 도형 그리기 (화살표, 선, 영역)
3. 세션 생성 및 저장/불러오기
4. 로컬 저장소 연동

### Phase 2
1. 애니메이션 기능
2. 포메이션 템플릿
3. 이미지 내보내기

### Phase 3
1. 고급 도형 편집
2. 텍스트 주석
3. 키보드 단축키
4. 성능 최적화

### 향후 기능
1. 협업 기능 (실시간 공유)
2. 클라우드 저장
3. 모바일 앱
4. 다른 스포츠 지원
5. 3D 뷰

## 제약사항

- **로컬 저장만 지원**: 초기 버전은 클라우드 저장 미지원
- **단일 사용자**: 협업 기능 없음 (향후 추가)
- **2D만 지원**: 3D 뷰는 향후 기능
- **웹 우선**: 모바일 최적화는 기본 수준만 제공
